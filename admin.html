GitHub upload

Uploads your APK to your GitHub repository using a temporary Personal Access Token (classic or fine-grained). The token is used only in your browser and is not stored.

Never reuse tokens. Create a short-lived token with minimum repo:contents permissions.

Permissions needed: repo (contents: read/write) or fine-grained: contents read/write for this repo.
